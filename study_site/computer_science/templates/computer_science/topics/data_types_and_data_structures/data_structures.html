Data Structures

Linked Lists

  • Similar to lists
  • Each item has data and a pointer

Tree

  • Used to represent hierarchical data
  • Each node is connected to other nodes
  • Nodes with no children are leaves
  • Nodes with no parents are roots

Binary Trees

The same as a regular tree but can only have up to two children.

Traversal

Depth First Search

Travel down the left hand path until you reach a leaf

Then head back up the hierarchical structure until you can go down down the tree again.
