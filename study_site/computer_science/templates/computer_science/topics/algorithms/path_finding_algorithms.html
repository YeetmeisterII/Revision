Path Finding Algorithms

Dijkstra's Algorithm

A shortest path algorithm.

Will always find the shortest path between two points on a network, it is not the most efficient method of path finding but will always find the shortest path with certainty.

  1. Pick an starting node and ending node
  2. Assume all nodes except from the ending node have a distance of infinity from the ending node, the ending node has a distance of zero from the ending node
  3. Now pick the node \(N\) with the lowest distance from the starting node
  4. Look at a node around it, call it \(M\). If the shortest route of \(N\) plus the distance of edge \(MN\) is less than the shortest route of \(M\), set \(M\)'s shortest route equal to it
  5. Repeat this process for every node connected to \(N\)
  6. Once you have done this mark off \(N\) as visited and repeat the process of picking the node with lowest distance from the ending node, however exclude nodes marked off as visited from being pick or having it's edges checked when looking at the nodes around a node being visited
  7. Repeat the process until you have visited the ending node
