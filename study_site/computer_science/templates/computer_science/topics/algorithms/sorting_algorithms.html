Sorting Algorithms

Insertion Sort

  1. Assume the left side of the list is sorted
  2. Look at the first and second item. If the second item is bigger do nothing, if smaller switch them
  3. Repeat this process with the second and third items but if they switch then compare the first item with the third to see if they should switch
  4. Repeat this process until the end of the list at which point it shall be sorted

Bubble Sort

  1. Starting with the first and second number
  2. If they're in the right order, do nothing. If the order is wrong, swap them and record that you made a swap
  3. Then move on to the second and third numbers
  4. Repeat the previous two actions until the end of the list
  5. If the number of swaps is not 0, repeat the iteration over the whole list
  6. When the number of swaps is 0 after an iteration over the list it is sorted

Merge Sort

  1. Split a list of \(N\) items into \(N\) lists of \(1\) item
  2. Take the first two lists and merge them with the lowest value being placed first and the higher of the two being placed second
  3. Repeat this for each pair of lists with one item until you only have lists with two items
  4. For each pair of lists with two items in them. Compare the first item in each list and add it to the next list
  5. Compare the first item in the two lists again (It's important to note that one of the lists will have one item in it at this point)
  6. Do this until all 4 items are in the new list
  7. Repeat the process building up longer and longer lists until you have a list of the original length

Quick Sort

  1. Pick a random number to be the pivot
  2. A pointer on the left looks for the first value that is greater than or equal to the pivot
  3. A pointer on the right looks for the first value that is less than the pivot until it reaches the pivot
  4. If both pointers are on the pivot then you have found the pivot's final position, else swap the positions of the two values with pointers
  5. Repeat the process picking a new pivot every time a pivot's final position is found until the list is ordered
