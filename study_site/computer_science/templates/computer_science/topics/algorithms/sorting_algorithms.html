Sorting Algorithms

Insertion Sort

Bubble Sort

Starting with the first and second number.

If they're in the right order, do nothing. If the order is wrong, swap them and record that you made a swap.

Then move on to the second and third numbers.

Repeat the previous two actions until the end of the list.

If the number of swaps is not 0, repeat the iteration over the whole list.

When the number of swaps is 0 after an iteration over the list it is sorted.

Merge Sort

Split a list of \(N\) items into \(N\) lists of \(1\) item.

Take the first two lists and merge them with the lowest value being placed first and the higher of the two being placed second.

Repeat this for each pair of lists with one item until you only have lists with two items.

For each pair of lists with two items in them. Compare the first item in each list and add it to the next list.

Compare the first item in the two lists again (It's important to note that one of the lists will have one item in it at this point).

Do this until all 4 items are in the new list.

Repeat the process building up longer and longer lists until you have a list of the original length.
