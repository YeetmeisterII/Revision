Lexical and Syntax Analysis

Stages of Translation

  1. Lexical Analysis performed by a lexer
  2. Syntax Analysis performed by a parser
  3. Semantic Analysis performed by a compiler
  4. Optimisation
  5. Code generation

Lexer

Manages the lexical analysis of the code.

  • Performs TOKENISATION
  • Breaks each line of code into a series of tokens
  • Tokens could be:
  • Identifier
  • Symbol
  • Operator
  • Keyword
  • Separator
  • Literal
  • Comment

Parser

Manages the syntax analysis of the code

  • Checks that tokens appear in the correct order.

Note: The parser does not check the datatypes.

Semantic Analysis

  • Checks that identifiers are created before they are used.

Symbol Table

Once everything has been lexed, parsed and analysed, the compiler generates a symbol table.

It contains information on the symbols' name, datatype, scope and (sometimes) memory address.
