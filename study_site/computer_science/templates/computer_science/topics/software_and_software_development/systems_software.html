Systems Software

Assembly Language

  • HLT (halt) - Stops program
  • ADD (addition) - Adds data at address to value in accumulator
  • SUB (subtract) - Subtract data at address from value in accumulator
  • LDA (load data) - Loads data from address into the accumulator
  • STA (store accumulator) - Stores data in the accumulator at the memory address
  • BRA (branch always) - Sets program counter to value (branch always)
  • BRZ (branch if zero) - Sets program counter to value if value in accumulator is zero
  • BRP (branch if positive) - Sets program counter to value if value in accumulator is positive
  • INP (input) - Input into accumulator
  • OUT (output) - Output contents of accumulator
  • DAT (data) - Specifies memory addresses used for data

Addressing

  • Immediate Addressing - Using the operand as the data to be passed
  • Direct Addressing - Operand refers to a memory address with data
  • Indirect Addressing - Operand refers to a memory address that itself refers to a memory address of data
