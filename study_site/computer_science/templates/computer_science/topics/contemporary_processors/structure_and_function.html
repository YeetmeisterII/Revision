Structure and Function

CPU

Performs most of the calculations for your computer.

  • Made up of billions of transistors
  • The transistors are arranged together to form logic circuits

CPU Performance

  • Clock Speed - How many times per second the CPU cycles
  • Cache Size - How much "on chip" memory there is

Parts of the CPU

  • ALU - Arithmetic Logic Unit
  • ACC - Accumulator (Register for the ALU)
  • CU - Control Unit
  • MAR - Memory Address Register
  • MDR - Memory Data Register
  • CIR - Current Instruction Register
  • PC - Program Counter
  • CB - Control Bus

Control Bus

Interrupts

An interrupt is when a device or program requests access to the CPU, the CPU responds by:

  • Suspending activities after current cycle
  • Saving the state of the registers
  • Calling the interrupt handler

Harvard Architecture

By using a different memory for instructions and data the rate of transfer in the buses can be faster by getting the data and instructions simultaneously.
