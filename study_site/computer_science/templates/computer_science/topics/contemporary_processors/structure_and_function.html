Assembly Language

Assembly language is a one-to-one mapping of commands to inbuilt processor commands.

The basic commands are:

  • ADD - Add a value in memory to the value in the accumulator.
  • SUB - Subtract the value in the accumulator by a value in memory.
  • HALT - End the running of the program.
