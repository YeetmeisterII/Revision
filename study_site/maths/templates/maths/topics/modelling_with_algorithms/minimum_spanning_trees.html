Minimum Spanning Tree

A minimum spanning tree is the minimum connected route (the least weight) that includes every node. If a graph has \(N\) nodes then the minimum spanning trees will have \(N - 1\) edges. Two of the algorithms we use to find this tree are Kruskal's and Prim's. Both have quadratic complexity however for large graphs Kruskal's is more efficient.

Kruskal's Algorithm

  1. Select the shortest arc
  2. Choose another arc which should not join nodes which are already connected to each other (in some way)

Prim's Algorithm

  1. Choose a node
  2. Build up a series of edges connecting the nearest node to the nodes that are currently connected
