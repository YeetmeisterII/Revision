Network Flows

Usage

When a continuous flow is required, the problem can ve modelled as a network flow. Weights on the arcs show capacities and the maximum continuous flow from start ot finish is required.

Terminology

  • There is a source node \( S \), essentially the starting node
  • There is a sink node \( T \), essentially the finishing node
  • A cut is a partition of the set of vertices into two sets (the source and sink must be in different sets)

Cuts

The capacity of a cut is the sum of all the capacities of the arcs which join the two sets
