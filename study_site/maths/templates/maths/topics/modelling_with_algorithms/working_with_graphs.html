Graphs

Terminology

Graph

A graph is a set of nodes (vertices) together with a set of arcs (edges).

Loop

An edge with the same vertex at both ends.

Simple Graph

A graph with no loops and no repeated edges.

Connected Graph

There must be a direct or indirect path to every other node.

Tree

A graph with the minimum edges used.

Network

A graph with weighted edges.
