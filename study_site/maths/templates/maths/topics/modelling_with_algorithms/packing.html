Packing

Packing is a non-trivial problem that is attempting to fit object of various "weights" into bins that have a maximum amount of "weight" they can store.

First Fit

  1. Take the first object in the list and place it into the first bin that can it can be fit into
  2. Repeat this process with the next objects in the list until all items are in a bin

First Fit Decreasing

  1. Sort the list into decreasing order
  2. Now perform the first fit method upon the sorted list
